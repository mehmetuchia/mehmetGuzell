Web Tasarımı

Web tasarımı bir web sitenin arama motorlarına erişebilirliğine yarayan, ana hatlarıyla kişi ile kurumları, hizmet veya ürünleri tanıtan grafikle metinleri bir araya gelerek kalite bir çalışma oluşturmasıdır. Web site tasarımları yapılırken temel giriş olarak bazı ana başlıklar vardır. Bunlar, html, domain, css, dinamik ile statik web sitesi, hosting, FTP’dir. Profesyonel web tasarım sitesi hazırlamak uzmanlık gerektiren iştir. Web sitesini hazırlamaya başlanılmadan önce bazı gerekli çalışmalar yapmak ve web sitesini zihinde veya kâğıt üzerinde tasarlaması yapılmalıdır.
